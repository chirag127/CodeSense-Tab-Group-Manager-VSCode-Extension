# CONTRIBUTING
## Introduction
Thank you for considering contributing to TabGroupSaver-Session-Manager-VSCode-Extension! This project is maintained by chirag127, and we welcome contributions from the community.
## Getting Started
1. **Fork the repository**: Create a fork of https://github.com/chirag127/TabGroupSaver-Session-Manager-VSCode-Extension to your GitHub account.
2. **Clone the repository**: Clone the forked repository to your local machine using `git clone https://github.com/your-username/TabGroupSaver-Session-Manager-VSCode-Extension.git`.
3. **Install dependencies**: Install the required dependencies using `npm install` or `yarn install`.
4. **Build and test**: Build the project using `npm run build` and test it using `npm run test`.
## Contribution Guidelines
* **Code style**: Follow the standard TypeScript coding conventions and the project's existing code style.
* **Commit messages**: Use descriptive commit messages that follow the Conventional Commits format.
* **Pull requests**: Create a pull request against the main branch with a clear description of the changes made.
* **Issues**: Report any issues or bugs found in the project using the issue tracker.
## Development Environment
* **VS Code**: This project uses VS Code as the primary development environment.
* **Extensions**: The following extensions are recommended for development:
    + TypeScript
    + ESLint
    + Prettier
* **Settings**: The project uses the following settings:
    + `typescript/tsdk`: `node_modules/typescript/lib`
